C'est quoi le Bus Factor ?

Le Bus Factor est le nombre minimum de personnes qui doivent quitter l'équipe avant que le projet soit bloqué.

Bus Factor = 1 : Une seule personne connaît le code → 🔴 Risque critique
Bus Factor = 2 : Deux personnes → 🟡 Risque moyen
Bus Factor ≥ 3 : Trois personnes ou plus → 🟢 Risque faible
