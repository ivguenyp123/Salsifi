🛡️ CIS GitLab Benchmark

CIS (Center for Internet Security) est un organisme indépendant qui publie des standards de sécurité reconnus mondialement.

Le CIS GitLab Benchmark v1.0.1 (avril 2024) est la référence officielle pour sécuriser GitLab en mode supply chain : 298 pages, 122 recommandations couvrant code source, CI/CD, dépendances, artifacts et déploiement.

📊 Ce module scanne:
• Section 1.1 - Code Changes (branch protection, approvals, signed commits)
• Section 1.2 - Repository Management (SECURITY.md)
• Section 1.3 - Contribution Access (maintainers, MFA)
• Section 1.4 - Third-Party (webhooks sécurisés)

Pourquoi c'est important ?

  • Standard international accepté par les auditeurs
  • Gap analysis objectif de votre posture de sécurité
  • Documentation compliance clé en main
  • Couverture supply chain complète

Ce module implémente actuellement 13 checks CIS. Le benchmark complet contient 122 recommandations.
